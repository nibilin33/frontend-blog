# 运行时的一些概念

# 帧

一个帧是一个连续的工作单元。当一个函数被调用时，运行时环境就会在栈中
创建一个帧。帧里面保存了特殊函数的参数和局部变量。

# 栈

栈包含了一个信息在执行时的所有步骤（帧）。

# 队列

队列包含一个待执行信息的列表，每一个信息都与一个函数相互联系。

# 堆

堆是一个内存存储空间，它不关注内部储存的内容的保存顺序。
堆中保存了所有正在被使用的变量和对象，同时也保存了一些当前
作用域已经不在被用到，但还没被垃圾回收的帧。
