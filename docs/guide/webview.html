# 移动端认证流程

1.内部项目：拿着应用ID向终端获取code，我们通过code向服务器换取token以及用户信息
2.工作台（集成第三方应用），所有信息都通过clientId向服务器获取重定向地址然后直接跳转

# 联调注意事项

2.移动端文件的相关同样需要调用他们的接口
